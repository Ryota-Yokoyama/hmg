## Handmade Garden
### Handmade Gardenとは
ハンドメイドに関するイベントを管理するアプリケーションです。
このアプリの利用ユーザの種類は下記の３種類を想定しています。
* イベントの主催者
  * ハンドメインメイドイベントの作成。
* イベントの出店者 (作家)
  * 主催者のイベントに出店する
* イベントの参加者（一般人）
  * ユーザ登録していないその他のユーザ。イベントの検索ができる。

### アプリケーション機能一覧
* ログイン・ログアウト機能（アクティベーションあり）
* プロフィール編集機能
* イベント検索機能
* イベント応募(参加)機能
* イベント作成機能
* イベント出店情報機能
* 参加イベント一覧機能
* イベントグループチャット機能
* ユーザ間メッセージチャット機能
* ユーザ検索機能
